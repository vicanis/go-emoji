# go-emoji

Emoji library for Golang. Emoji list was taken from **https://apps.timwhitlock.info/emoji/tables/unicode**

Usage:

```
package main

import (
    emoji "github.com/vicanis/go-emoji"
)

func main() {
    log.Print(emoji.WavingHandSign + ", world!")
}
```

Returns

```
👋, world!
```
